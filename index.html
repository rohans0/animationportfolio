animation system within game engine

live web demo
source code

Group project and final project for game engine development class, an animation system added to a custom WebGL game engine (a majority of the class was understanding/coding parts within the engine rather than making it entirely).

It linearly interpolates between keyframes of object position, size, rotation, and color, and was inspired by Godot's animation system which allows linear interpretation (along with other types like frame-by-frame) of most properties of objects by choosing key frames at times.
